## Notes for Using This Template

1. **Number sequentially** - ADR-001, ADR-002, etc.
2. **Use Y-statement format** for the Decision section
3. **Keep it concise** - ADRs should be scannable
4. **Link to context** - Reference related docs
5. **Update status** - Mark deprecated/superseded when things change
6. **Date format** - Use DD/MM/YYYY (UK format)
